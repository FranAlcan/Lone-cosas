Resumen de notas - Laboratorio Criptografía en Linux

1. Objetivo del laboratorio:
   - Practicar tareas básicas de criptografía en Linux.
   - Usar cifrado César y OpenSSL para recuperar archivos cifrados.

2. Escenario:
   - Directorio personal está cifrado.
   - Recuperar archivos descifrando mensajes ocultos.

3. Herramientas y comandos usados:
   - ls: listar archivos.
   - cat: mostrar contenido de archivos.
   - cd: cambiar directorio.
   - ls -a: mostrar archivos ocultos.
   - tr: traducir caracteres (descifrado César).
   - openssl aes-256-cbc: descifrar con una clave.

4. Proceso general:
   - Buscar pistas en README.txt.
   - Entrar a subdirectorio "caesar".
   - Hallar archivo oculto .leftShift3.
   - Aplicar tr para descifrado César (shift 3 izquierda).
   - Obtener comando para descifrar archivo principal.
   - Ejecutar comando OpenSSL para recuperar datos.

5. Comando importante revelado:
   openssl aes-256-cbc -pbkdf2 -a -d -in Q1.encrypted -out Q1.recovered -k ettubrute

6. Archivos involucrados:
   - Q1.encrypted → archivo cifrado inicial.
   - .leftShift3 → mensaje cifrado César.
   - Q1.recovered → archivo final descifrado.

7. Resultado final:
   - Se logró descifrar mensajes, archivos ocultos y archivos cifrados.
